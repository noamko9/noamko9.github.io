Installation Guide for LSTM Accelerator

Installation

Go to: Xilinx Vivado Download Page and install version 2022.2.

Step-by-Step Guide

Building the Project in HLS

Open vitis HLS cmd and run the pre-prepared scripts to rebuild the accelerator. The LSTMAccel folder will be created.

The folder LSTMAccel is also located in ../../vivado_project/proj_ips/ip.

Moving to Vivado

Open Vivado, navigate to the vivado_project folder and open the recreate_tflm_vivado project. Click on "Generate Bitstream" to get the output file.

The Vivado output is a file with the .xsa extension.

Final Step in Vitis

Select the folder ..\NNAccel-main\Vitis\vitis_workspace as input. Run create_platform.tcl to create the platform.
