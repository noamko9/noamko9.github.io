Rebuilding the Project

Start by building the project in HLS:

  • Open `vitis HLS cmd` and run pre-prepared scripts to rebuild the accelerator according to the README file in the folder.
  • A folder named `LSTMAccel` will be created. It can also be found on GitHub for process simplification, but it can be built manually.
  • The `LSTMAccel` folder is also saved in `../../vivado_project/proj_ips/ip`.

Using Vivado

  • Open Vivado software and navigate to `vivado_project`.
  • Open the `recreate_tflm_vivado` project and click "Generate Bitstream".
  • The result is a file with the `.xsa` extension.

Final Step in Vitis

Select the folder `..\NNAccel-main\Vitis\vitis_workspace` as input. Run `create_platform.tcl` to create the platform. Then, you can rebuild the applications using the remaining scripts.
